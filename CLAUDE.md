# Projekt: Silke Burkhardt – Website Redesign

## Überblick
Professionelle One-Pager Website für Silke Burkhardt – Systemische Beraterin, Supervisorin und Pädagogin aus Österreich.

## Tech-Stack
- **Framework:** Vite + React 18
- **Styling:** Tailwind CSS 3
- **Sprache:** Deutsch (de-AT)
- **Deployment:** Statisches Hosting (Vercel/Netlify-ready)

## Design-System
- **Stil:** Warm & Organisch – keine generischen Templates
- **Primary:** #C4704B (Terrakotta)
- **Secondary:** #8B9E7C (Salbeigrün)
- **Background:** #FAF7F2 (Cremeweiß), #E8DFD3 (Sand)
- **Text:** #2D2926 (Warmes Dunkelbraun)
- **Akzent:** #B8975A (Gedämpftes Gold)
- **Fonts:** Inter (Body), Playfair Display (Headlines)

## Sektionen
1. Hero – Claim + Titel
2. Über mich – Persönliche Vorstellung
3. Leistungen – 6 Service-Cards
4. Arbeitsweise – Systemischer Ansatz
5. Kontakt – Formular + Kontaktdaten

## Konventionen
- Komponenten in `src/components/`
- Tailwind-Utility-First, custom Klassen nur wenn nötig
- Deutsche Texte, professioneller Ton
- Mobile-first responsive Design
- Keine externen UI-Libraries – alles handcrafted
